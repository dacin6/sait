# Запрошення в ботанічний сад — маленький сайт

Прості файли: `index.html`, `styles.css`, `script.js`.

- Відкрий `index.html` у браузері.
- Кнопка "Ні" хитра — відходить від курсора.
- При натисканні "Так" можна вибрати дату; дата зберігається у `localStorage`.

Можна адаптувати тексти, кольори і додати бекенд для реального RSVP.
